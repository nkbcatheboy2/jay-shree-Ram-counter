# 🚩 SoundApp - Modern Text-to-Speech Repeater

**SoundApp** ek powerful aur stylish single-page React application hai jo browser ki built-in **Web Speech API** ka use karke text-to-speech functionality provide karta hai. Isme user-defined repetition logic, activity history, aur ek premium **Dark Glassmorphism** UI diya gaya hai.

---

## ✨ Key Features

- 🔊 **Infinite Voice Loop:** Aap jo bhi text input karenge, app use aapki set ki gayi limit tak repeat karega.
- 🕒 **Activity Tracking:** Ek dynamic sidebar jo aapki voice commands ki history (Text, Count, aur Timestamp) ko track karta hai.
- 👤 **User Dashboard UI:** Navigation bar ke saath personalized welcome message aur admin badge.
- 🛑 **Instant Stop:** Kisi bhi waqt voice loop ko beech mein rokne ke liye dedicated stop button.
- 🎨 **Glassmorphism Design:** Modern Dark Theme jisme blur effects, neon glow, aur smooth animations ka use kiya gaya hai.

---

## 🛠️ Tech Stack

- **Frontend:** React.js (Hooks like `useState`, `useEffect`)
- **Styling:** CSS3 (Modern Glassmorphism, CSS Gradients, Flexbox/Grid)
- **API:** Web Speech Synthesis API (Native Browser Support)
- **Icons/Fonts:** Poppins Font, Emoji Icons
